上課記錄

week1:重新出發

By Hannah 13 March 2022

開課前

這週是上課的第一週

在上課之前，訓練單位都會要求學員要通過 python 的先修課程(總共 18 小時)，採用線上的方式進行，不過是預錄的影片

18 小時的課程，要在考試前自行分配進度看完

覺得某種程度上也是在考驗人的自律能力阿~

考試也是採線上的方式進行，題目不難，皆是選擇題，基本上有認真看影片不會有太大的問題

通過後就會收到下面的簡訊囉~~

開課第一天

第一天基本上沒有上課

主要都是在講解班規和完成一些班級庶務

剛好我的位子上的滑鼠跟我家裡的一樣呵呵

上課第一週

基本上第一週主要都在介紹 HTML 的各個屬性和標籤的應用，也稍微帶到 CSS 和 JS 的概念，但比重不多，大概 3 成

每天回到家也一定會複習上課所學，把 code 再實作一次

瀏覽官方文件，把標籤和屬性的用意看懂

其實就好像在組裝機器人，慢慢地拼湊頭、身體、腳

師資的部分

目前最大的收穫是老師對於一件事的思考方式，層層拆解程式碼的意思，最後再來個完美生動的比喻

不禁讓人讚嘆，工程師的邏輯思考是多麼的縝密

此外，老師會引導你如何下關鍵字在搜尋引勤查找資料(廢話…誰不會)

這裡的意思是說如何找到 更精確的資料，有問題也不會直接告訴你答案，迫使你必須靠自己來解決問題，

在這套用一句《老子》裡的一段話 授人以魚不如授人以漁，

習慣直接得到答案，就少了思考的步驟，那會很可惜~思考不僅重要也幫助長期記憶，

總結

第一週都有跟上老師的上課步調，下一週繼續加油~

Hannah 筆
