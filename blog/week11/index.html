Week11-RESTful API + 生日週

By Hannah 27 May 2022

遠距教學邁入第二周

已經慢慢能適應線上上課的模式，也開始漸漸感受到其便利性和優點

這週上了 HTML5 API 和 RESTful API

這週來紀錄 RESTful API

RESTful API

RESTful API 上了 1.5 天的時間 本週進度簡述： XMLHttpRequest、DOM XML Parser、openData 例子練習(氣象局)、$.get/$.post/$.ajax、promise、JavaScript async 非同步/await 等

因為上課時間較緊湊的關係，所以教得較快，在課後的複習上，因此花了比平常更多的時間去熟悉這禮拜的課程內容，Promise 的部分和 async/await 個人覺得比較難，這是在非同步上很重要的知識點

專題實作

這週我們把專題的草稿圖交了出去，網站名稱叫做「宇你相遇」，命名上想讓人一看就知道我們在做什麼，藉此去呼應交友軟體與人相遇的特點，並利用「與」的諧音去呼應我們的網站風格！另外還畫了 logo 草圖供組員參考

草圖是在生日當天畫的，那天下著大雨，原本跟朋友約好 18:00 吃飯，不過因為雨勢很大所以朋友因此就比較晚到，因為餐廳離家很近，所以我先在約定的時間內抵達了餐廳，待服務生引領到座位後，便拿起包包裡的平板和觸控筆，開始把心中的構想畫下來，也參考了一些跟外太空有關的圖，也算是自己給自己的生日禮物吧哈哈哈！後來草圖未被採納，因為也有別組做很類似的東西~

專題發表需要做的準備

另外老師也說明了專題發表當天我們該準備好什麼，除了要做好簡報以外，還要做一份企劃書，裡面要有市調，例如：目標客群分析、競爭者分析、SWOT 等和網站設計與建置，包含網頁色彩配置、LOGO 商標設計，還要有網站各個頁面的架構說明、設計流程圖 Flow Chart/UI Flow 等

一下接收了好多資訊，腦袋還處於混亂的狀態，得花點時間消化整理一下老師所講的東西，不過自己好像被制約一樣，講到專題心中的壓迫感又出現了，難以描述的感覺，但我想這似乎是自己對壓力的生理反應

專題結束後，也別忘了好好休息

下禮拜各組開始要與老師來討論這禮拜教出去的專題草稿
