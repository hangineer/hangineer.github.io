Week12-疫情高峰 啟動線上課程

By Hannah 20 May 2022

遠距教學的啟動

由於疫情越來越嚴峻，所以這週便開始啟動線上教學。

線上更需要自律能力

實體和線上比對下來，如果你夠自律的話，我覺得線上的優點是多於實體的，各有好壞。實體課程比較能夠感受到學習的壓力，也比線上課程多了份臨場感，因為在教室裡你看得到同學老師，人嘛~眼見為憑，當你發現大家其實都很努力也很認真時，自然就不會太鬆懈，會有股壓力在推著你前進。

簡要本週上課內容

來談談這週的上課內容好了，這週都在講 HTML5API 的部分，一開始對 API 的概念感到十分的抽象，什麼是 API？什麼是 API 的串接？

網路上有很多淺顯易懂的影片 在此附上我覺得不錯的

https://youtu.be/s7wmiS2mSXY英文版

https://youtu.be/zvKadd9Cflc中文版

然後，老師還跟我們談了: 購物車的建置（可以算出各個欄位的小計和全部總價）、表單驗證、為影片上字幕、拖拉事件、class、建構子、HTML5 新增的標籤 Canvas 等等

雖然課表上是寫 HTML5，但實際上用到比較多 JavaScript ，最後在老師的帶領下，利用 Canvas 做出了類似小畫家的畫畫面板，可以把你所畫的圖下載成圖片格式儲存，並不會很困難，但對於前端新手來說就像劉姥姥進大觀園一樣感到新奇無比。

不知不覺又到了深夜

本週的記錄就先到這裡
