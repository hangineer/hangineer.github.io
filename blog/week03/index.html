week03-感受css的神奇

By Hannah 26 March 2022

CSS

這週主要是上 CSS

總共 8 堂課，24hr

第一週在講 HTML 的時候，有稍微講到 CSS 的部分

所以這週上起來輕鬆許多

但 float、flexbox、RWD、grid System

比較需要花時間多練習

個人覺得排版是蠻重要的一部分

了解切版的精神，再去使用框架

適應

這週有比較適應上課的步調

老師有在 Google Classroom 公告作業

讓大家練習切版和 RWD

利用@media query 設定斷點,響應各種裝置大小不同的需求

但 RWD 的效果一直做不好,感到些挫折

上網爬了很多文章，問了一下本科系的朋友後

足足弄了一整天才解決!!

原來是自己一開始就把圖片的大小給固定了

不過從中發現自己是對 RWD 不夠熟悉的

這也代表之後做專題可能也會遇到類似的問題

把握每次練習的機會,從中學習解決問題的能力

相信能越來越能掌握某些技巧

總結

css 讓網頁變得更有生命力

我覺得是到目前為止最喜歡的課程^^

在做作業的過程中

雖然尋找問題就花了大半的時間

但問題迎刃而解時

成就感便從心中油然而生

記住這個感覺

並繼續努力吧~

Hannah 筆
